# RVT Documentation

Documentation of the RVT ecosystem.

The documentation uses vuepress (https://vuepress.vuejs.org/). To build the
system, install npm or yarn and, from this directory, run:

```
yarn: yarn install && yarn run dev
npm: npm install && npm run dev
```

# First configuration

Some packages are created automatically from the RVT2 documentation.
See the initial lines in `autodoc.sh`. You must run this file lines to
recreate these packages
